# User Info Mini Project

This is a simple Python mini project that takes user input and displays formatted information.

## 📌 Features
- Takes user name, age, and city as input
- Determines whether the user is an adult or minor
- Displays user details in a clean format
- Calculates approximate birth year

## 🧠 Concepts Used
- Input and Output
- Type Casting
- Conditional Statements (if-else)
- Variables

## ▶️ How to Run

1. Open terminal
2. Run the file:

```bash
python user_info.py
